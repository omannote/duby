# Laundry POS

This project provides a basic structure for a laundry POS system using PHP and MySQL.
The `database/schema.sql` file contains the initial schema and demo data.
Place the contents of the `public` folder in your web server root.

After logging in, access `/pos.php` to create new orders using the POS interface.
